# Programming for Data Analysis - 小组作业

## 课程作业标题

**网络攻击事件分析 (Analysis of Cyber-Attack Events)**

---

## 概述

在本次作业中，你将与团队成员协作分析一系列与网络攻击事件相关的数据集。你需要使用 R 语言的数据分析技术对这些数据集进行合并、清洗和探索，以发现有意义的洞察和趋势。

你的分析应当：
- 识别数据质量问题
- 检测数据模式
- 生成可操作的洞察

作业必须展现出超越课堂示例的深度和原创性。你需要应用数据探索、转换、可视化和统计分析等概念，并为每个分析决策提供清晰的理由说明。

**重要要求**：
- 所有任务必须完全使用 R 和 RStudio 执行，包括数据预处理、分析和可视化
- 不允许使用外部工具（如 Excel、Power BI 等）

最后，你需要在一份结构完善的分析报告中记录你的整个过程、分析和发现，清晰解释每个步骤及其理由。

---

## 作业目标

本次作业旨在使你能够：

1. **应用 R 编程和数据分析技术**：清洗、整合和分析真实世界数据
2. **发展数据转换能力**：将原始数据转化为有意义且可解释的结果，支持决策制定
3. **提升协作技能**：通过完成团队分析项目增强合作能力

---

## 作业类型

**小组作业**（每组最多 4 人）

## 数据集描述

数据集包含有关网站篡改事件的信息。每条记录包括：

### 事件详情
| 字段 | 描述 |
|------|------|
| Date | 篡改日期 |
| Notify | 报告来源（通知方） |
| URL | 受影响的网址 |

### 技术属性
| 字段 | 描述 |
|------|------|
| IP | 服务器 IP 地址 |
| Country | 托管国家 |
| WebServer | Web 服务器版本 |
| Encoding | 字符编码 |

### 影响变量
| 字段 | 描述 |
|------|------|
| Ransom | 勒索金额（千美元） |
| DownTime | 停机时间（天） |
| Loss | 预估经济损失 |

### 数据文件

你将收到三个数据集（在 Moodle 上），需要进行合并、清洗和准备：

| 文件名 | 格式 | 大小 |
|--------|------|------|
| HackingData_Part1 | CSV (逗号分隔值) | 20,046 KB |
| HackingData_Part2 | Excel 工作表 | 15,480 KB |
| HackingData_Part3 | 文本文档 | 17,894 KB |
| HackingData Dataset Description | 文本文档 | 1 KB |

## 一般要求

1. **数据驱动分析**：分析必须有意义且基于数据，通过证据支持决策
2. **工具限制**：所有数据预处理、分析和可视化必须完全使用 RStudio 中的 R 脚本执行
3. **禁止外部工具**：不允许使用 Excel、Power BI、Tableau 等外部工具
4. **良好编程实践**：
   - 清晰的注释
   - 一致的变量命名
   - 适当的缩进
5. **背景研究**：可以进行额外的背景研究以增强对数据集和领域的理解
6. **结构化方法**：展示结构化的分析方法，确保可视化能有效传达洞察
