# Project Documentation Folder

This folder contains all project reports and documentation files.

## Folder Structure

```
docs/
├── README.md                          # This file
├── outlier_detect.md                  # Outlier detection methodology documentation
├── DATA_CLEANING_PIPELINE_SUMMARY.md  # Data cleaning pipeline summary
├── DATA_CLEANING_REPORT_TEAM_SUMMARY.md # Team data cleaning report summary
├── EXECUTIVE_SUMMARY_ONE_PAGE.md      # Executive summary (one-page version)
├── IMPUTATION_VALIDATION_SUMMARY.md    # Imputation validation summary
└── reports/                           # Reports subfolder
    ├── *.pdf                          # All PDF visualization reports
    └── *.csv                          # All CSV statistical reports
```

## Report Files Description

### PDF Visualization Reports
- `distribution_diagnostics.pdf` - Distribution diagnostic plots

- `multivariate_outlier_analysis.pdf` - Multivariate outlier analysis
- `post_cleaning_distribution.pdf` - Post-cleaning distribution plots
- `final_association_check.pdf` - Final association check
- `final_missing_patterns.pdf` - Final missing patterns
- `imputation_validation_plots.pdf` - Imputation validation plots
- `before_after_comparison.pdf` - Before and after cleaning comparison

### CSV Statistical Reports
- `outlier_detection_summary.csv` - Outlier detection summary
- `outlier_removal_report.csv` - Outlier removal report
- `multivariate_outlier_summary.csv` - Multivariate outlier summary
- `multivariate_outlier_removal_report.csv` - Multivariate outlier removal report
- `categorical_cleaning_summary.csv` - Categorical variable cleaning summary
- `before_after_statistics.csv` - Before and after cleaning statistics comparison
